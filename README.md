# DevNest Portfolio

Десктопное приложение на Electron для создания портфолио разработчика через удобную админ-панель.  
Пользователь добавляет темы, проекты, описания, ссылки, изображения и фото профиля, а затем экспортирует всё в полностью статический сайт.

## О проекте

Portfolio App — это локальное приложение для быстрого создания портфолио без ручной верстки.  
Всё управление происходит внутри программы: вы создаёте темы, добавляете проекты, загружаете обложки, заполняете описание и экспортируете готовый сайт для публикации.

Проект подходит для:
- личного портфолио;
- учебных проектов;
- демонстрации работ;
- быстрого создания сайта без CMS.

---

## Возможности

- удобная админ-панель;
- создание, переименование и удаление тем;
- создание, редактирование и удаление проектов;
- загрузка обложек для проектов;
- добавление фото и био разработчика;
- локальное сохранение данных;
- экспорт в статический сайт;
- поиск по проектам в текущей теме;
- portable-сборка для Windows.

## Как запустить локально

### 1. Установить зависимости

```bash
npm install
```

### 2. Запустить приложение

```bash
npm start
```

---

## Как собрать portable-версию

### 1. Установить зависимости

```bash
npm install
```

### 2. Собрать portable exe

```bash
npm run dist
```

После сборки portable-файл появится в папке `dist`.

---

## Как работает экспорт

Приложение сохраняет данные локально и умеет экспортировать готовый сайт в отдельную папку.

При экспорте создаются:

* `index.html`
* `styles.css`
* `script.js`
* `data.json`
* изображения проектов
* фото разработчика

Готовый сайт можно открыть в браузере или загрузить на хостинг.

---

## Где хранятся данные

Локальные данные приложения, фото и обложки сохраняются в папке пользователя приложения.
Это позволяет не терять проекты после перезапуска программы.
